MÓDULO 3 INTRODUCCIÓN AL CSS

  1. Definición de CSS

    2. CSS, o Cascading Style Sheets, es un lenguaje de diseño utilizado para describir la presentación y el formato de un documento escrito en HTML o XML. CSS permite a los desarrolladores definir aspectos como el color, las fuentes, el espaciado y la disposición de los elementos en una página web. Su uso facilita la separación del contenido (HTML) de la presentación (CSS), lo que mejora la mantenibilidad y la flexibilidad del diseño web.

  2. Funcionamiento Básico de CSS

    3. Separación de la estructura y la presentación. CSS permite separar el contenido HTML de su presentación visual. Es decir, te permite mantener el código HTML limpio y estructurado, mientras que el estilo se define en un archivo CSS separado. Este orden mejora la legibilidad del código y facilita el mantenimiento, así como la actualización de los estilos.

    Consistencia y mantenibilidad. Al utilizar CSS, puedes aplicar estilos de manera consistente a un sitio o aplicación web. Los estilos se definen una vez y se aplican a múltiples elementos en las páginas, lo que favorece la renovación de la apariencia visual de todo el proyecto.

    Eficiencia en el rendimiento. CSS permite cargar estilos externos en un archivo separado. Con esto, el navegador almacenará en caché los estilos y los aplicará a todas las páginas del sitio, lo que mejora el rendimiento al reducir la cantidad de datos que deben transferirse entre el servidor y el cliente.

    Flexibilidad y control. Asimismo, ofrece una amplia gama de propiedades y selectores con los que tendrás un control preciso sobre el estilo de los elementos HTML. Podrás modificar, de forma sencilla, los colores, fuentes, márgenes, tamaños, diseños, etc. Gracias a esta característica, podrás personalizar y adaptar a varios dispositivos y tamaños de pantalla tu sitio web.

  3. Incluir CSS en un archivo HTML

    4. Existen tres formas principales de incluir CSS en un archivo HTML:

    1. CSS Interno

      2. Se incluye en la sección < head > del documento HTML dentro de un bloque < "style"> Este método es útil para estilos específicos de una sola página.

    2. CSS Externo

      3. Se utiliza un archivo CSS separado y se enlaza en la sección < "head" > del HTML con la etiqueta <"link">. Este método es ideal para mantener el código limpio y reutilizar estilos en múltiples páginas.

      4. Se aplica directamente a un elemento HTML utilizando el atributo style. Este método es menos recomendado para mantener la separación de estilos, pero es útil para estilos rápidos.

  4. Definición de una regla CSS

    5. Una regla CSS es una declaración que define cómo se deben estilizar uno o más elementos HTML. Consiste en dos partes principales: el selector y el bloque de declaraciones.

    Componentes de una Regla CSS:

    • Selector:

      • Especifica a qué elementos se aplicarán los estilos. Puede ser una etiqueta HTML, una clase, un ID o una combinación de ellos

    • Bloque de Declaraciones:

      • Se encuentra entre llaves {} y contiene una o más declaraciones de estilo. Cada declaración está formada por una propiedad y un valor, separados por dos puntos: Las declaraciones se separan con punto y coma;

  5. Elemento de una regla CSS
    • Regla

      • Una regla CSS es la combinación de un selector y un bloque de declaraciones. Define cómo se deben estilizar los elementos HTML seleccionados.

    • Selector

      • El selector especifica a qué elemento(s) HTML se aplicarán los estilos definidos en la regla. Puede ser:

      • Etiqueta: Por ejemplo, h1, p.
      • Clase: Por ejemplo, .clase.
      • ID: Por ejemplo, #id.
      • Combinaciones: Como div p, que selecciona < p > dentro de < div >.
    • Bloque de declaración

      • Es la parte de la regla que contiene las declaraciones de estilo. Se encuentra entre llaves {}. Dentro del bloque, se pueden incluir múltiples declaraciones.

    • Propiedad

      Una propiedad es un aspecto del estilo que deseas cambiar. Algunas propiedades comunes incluyen:

      • color: El color del texto
      • font-size: El tamaño de la fuente.
      • margin: El espacio exterior alrededor de un elemento.
    • Valor

      • El valor es lo que se asigna a una propiedad específica. Por ejemplo:

      • Para color, un valor podría ser red o #ff0000.
      • Para font-size, un valor podría ser 16px o 1em
  6. Medios CSS

    7. Una de las características más importantes de las hojas de estilos CSS es que permiten definir diferentes estilos para diferentes medios o dispositivos: pantallas, impresoras, móviles, proyectores, etc.

    Los medios más utilizados actualmente son screen (para definir el aspecto de la página en pantalla) y print (para definir el aspecto de la página cuando se imprime), seguidos de handheld (que define el aspecto de la página cuando se visualiza mediante un dispositivo móvil).

    Las reglas @media son un tipo especial de regla CSS que permiten indicar de forma directa el medio o medios en los que se aplicarán los estilos incluidos en la regla. Para especificar el medio en el que se aplican los estilos, se incluye su nombre después de @media. Si los estilos se aplican a varios medios, se incluyen los nombres de todos los medios separados por comas

    A continuación, se muestra un ejemplo sencillo:

    @media print { body { font-size: 10pt } }

    @media screen { body { font-size: 13px } }

    @media screen, print { body { line-height: 1.2 } }

  7. Comentarios CSS

    8. Los comentarios en CSS son útiles para agregar notas o explicaciones dentro del código sin que afecten la ejecución de los estilos. Esto ayuda a mantener el código organizado y comprensible, especialmente cuando se trabaja en proyectos más grandes.

  8. Sintaxis de la definición de las propiedades CSS

    9. Sintaxis General

    La definición de una propiedad CSS se compone de una propiedad y un valor, separados por dos puntos: y terminados con un punto y coma;. Todo esto se coloca dentro de un bloque de declaraciones, que está encerrado entre llaves {}.

    Componentes

    • Selector: Especifica el elemento HTML al que se aplicará la regla.
    • Propiedad: Es el aspecto del estilo que deseas modificar (por ejemplo, color, font-size, margin).
    • Valor: Especifica cómo se debe aplicar la propiedad (por ejemplo, red, 16px, 10%).
